# Simulador SLA Marketplace KAI

Aplicacion web estatica basada en el workbook `2026.03.26.SLA_simulador_Assessment.xlsx`.

## Incluye

- Inputs guiados para acuerdo proveedor-marketplace.
- Calculo de GMV, comision, fee, float financiero, penalizaciones, bono e ingreso neto.
- Comparativa objetivo vs real para indicadores SLA.
- Riesgo SLA, financiero y cliente.
- Decision ejecutiva GO / REVISAR.
- Resumen contractual generado automaticamente.

## Despliegue

Proyecto preparado para Vercel como sitio estatico. La entrada principal es `index.html`.
